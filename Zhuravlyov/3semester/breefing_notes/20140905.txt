Бот для шляпы.
Односторонний. Такое своеобразное приложение, рассказывающее, как лучше всего объяснять данное слово.

Есть 2 глобальных модуля:
I) выбор объяснений
II) отображение объяснений.

Точно продумать, какое взаимодействие между этими 2 модулями будут, заранее невозможно.
Нужно стараться сделать так, чтобы добавление каждого пункта было простым.
К следующей паре нужно составить список взаимодействий, которые могут быть.
Возможно, выбору объяснений нужно передавать информацию о длительности, сколько времени осталось и т.д.
Выбор объяснения должен знать объяснение, вероятность, с которой будет понято, информацию о времени...

Что есть глобальное в выборе объяснений?
1) Обучение: сбор данных для обучения, выбор, подсчёт
2) Хранение и отдача (взаимодействие с отображением)

А что у нас в результате возможно будет?
Будет код, который будет анализировать данные. Будет код, который взаимодействует с базой данных.
Будет API для взаимодействия. Будет web-страница, на которой можно посмотреть объяснение слова. Будет приложение.
Пока что общая сущность не продумана.

Способы объяснения слова (пока придуманные)
1) Определение
2) Синонимы
3) По частям
4) По созвучным словам
